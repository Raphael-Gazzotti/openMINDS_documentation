Welcome to the openMINDS documentation!
===================================

**openMINDS** is a metadata framework for graph database systems. The framework comprises:

- **specifications** of schemas for multiple, interlocked metadata models
- **libraries** for serviceable, well-defined terms and constructs
- **supportive tooling** for handling openMINDS in Python or MATLAB

.. note::
   This project is under active development.
