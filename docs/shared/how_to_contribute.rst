#################
How to contribute
#################

The openMINDS metadata framework is an open-source project and contributions from the community are highly appreciated. You can contribute in many different ways. 

You can contribute by providing feedback and making suggestions through our issue trackers, or creating pull request for fixing issues and new developments.

.. note::
   All contributions to the openMINDS framework should be a harassment-free! Please follow our `code of conduct <how_to_contribute/code_of_conduct>`_ to guarantee an open and welcoming community environment.
